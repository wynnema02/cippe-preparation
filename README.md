# CIPPE 刷题练习

一个用于 CIPPE 备考的静态刷题网站，内置 289 道练习题，支持刷题、背题、错题复盘、错题重刷，以及通过 JSON 文件手动同步学习进度。

线上地址：[https://wynnema02.github.io/cippe-preparation/](https://wynnema02.github.io/cippe-preparation/)

## 功能

- 刷题模式：按题号顺序从未刷过的题开始练习，交卷后统计正确率。
- 背题模式：直接展开题目、答案和解析，适合快速复习。
- 错题回顾：集中查看错题、正确答案和解析。
- 错题重刷：重新练习错题，答对后自动移出错题本。
- 进度统计：显示已刷题数、累计答题次数和错题数量。
- 手动同步：导出或导入 JSON 文件，在不同设备之间迁移进度。

## 进度同步

网站会把学习进度保存在当前浏览器本地。换设备或换浏览器时，可以用页面里的“进度同步”功能手动迁移。

### 导出进度

1. 打开网站。
2. 点击“导出进度”。
3. 浏览器会下载一个类似 `cippe-progress-20260707T102300.json` 的文件。
4. 把这个文件保存到 iCloud、网盘、微信文件传输、邮箱或其他你常用的位置。

导出的内容包括：

- 已刷题目
- 累计答题次数
- 错题本
- 错题次数
- 最近一次错误答案
- 最近练习时间

### 导入进度

1. 在另一台设备或浏览器打开网站。
2. 选择导入方式。
3. 点击“导入进度”。
4. 选择之前导出的 JSON 文件。

导入方式有两种：

- 合并导入：默认选项。保留当前设备已有进度，并把 JSON 文件里的记录合并进来。
- 覆盖导入：清空当前设备已有进度，完全使用 JSON 文件里的记录。覆盖前页面会再次确认。

导入时会校验题号，只接受当前题库里存在的题目记录。

## 文件结构

```text
.
├── index.html
├── styles.css
├── app.js
└── data/
    └── questions.js
```

- `index.html`：页面结构。
- `styles.css`：页面样式。
- `app.js`：刷题逻辑、错题本、进度统计和 JSON 导入导出。
- `data/questions.js`：题库数据。


## 本地预览

可以在项目目录启动一个简单的静态服务器：

```bash
python3 -m http.server 8765
```

然后访问：

```text
http://127.0.0.1:8765/
```

## 数据说明

进度数据只保存在浏览器本地或你手动导出的 JSON 文件中。网站本身没有账号系统，也不会把个人刷题记录自动上传到服务器。
